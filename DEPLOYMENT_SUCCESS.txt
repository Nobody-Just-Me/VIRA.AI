🎯 NEXT STEPS FOR USERS
========================

1. Clone repository:
   git clone https://github.com/Nobody-Just-Me/VIRA.AI.git

2. Build APK:
   cd VIRA.AI
   bash build.sh

3. Install:
   adb install -r VIRA-Release.apk

4. Configure:
   - Open VIRA
   - Go to Settings
   - Enter Groq API key
   - (Optional) Enter ElevenLabs API key

5. Enjoy!
